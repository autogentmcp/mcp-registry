MCP Registry API

Welcome to the Model Context Protocol (MCP) Registry API. This service allows applications to register and discover MCP endpoints.

Key Endpoints

GET /applications

List all applications registered in the system. Requires admin authentication.

GET /applications/with-endpoints

List all applications with their endpoints for a specific environment. Requires admin authentication.

GET /endpoints

List all endpoints for an application in a specific environment. Provides environment data including baseDomain.

POST /endpoints

Register endpoints for an application. Authentication is done via API key.

About MCP

The Model Context Protocol (MCP) provides a standardized way for applications to communicate with AI models and exchange context information. This registry serves as a central directory for MCP-compatible endpoints, allowing applications to discover and interact with each other.
